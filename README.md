# 감정 분석 기반 상담 서비스

## 1. 백엔드 프로젝트 라이브러리
- ```pip install "fastapi[standard]" ```

- ```pip install fastapi uvicorn torch transformers opencv-python Pillow numpy```

- ```pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124```

<br>

## 2. 프론트엔드 실행
- ```npm i```

- ```npm run dev```
